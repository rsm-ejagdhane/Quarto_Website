Introduction

Note

Write a short introduction for the reader that sets the scene. Cover the following points in your own words:

  • You have a website and your landing page includes a place where visitors can sign up to receive your newsletter.

  • You want to test which of two possible “calls to action” (CTAs) leads to more sign-ups. CTA A reads “Sign up for our newsletter here!” and CTA B reads “Stay up to date by signing up!” (you are welcome to change the language of the CTAs).

  • You will evaluate which CTA performs better by running an A/B test. Explain what an A/B test is: visitors are randomly assigned to see one of the two CTAs, and you compare sign-up rates between the two groups.

The A/B Test as a Statistical Problem

Note

In this section, connect the A/B test to a formal statistical framework. Address each of the following ideas:

  • Each visitor either signs up or does not, so we can model each visitor’s outcome as a draw from a Bernoulli distribution with parameter π, the probability of signing up.

  • Because the CTA a visitor sees may influence their probability of signing up, we allow the parameter to differ across groups: visitors who see CTA A have sign-up probability πA, and visitors who see CTA B have sign-up probability πB. Each visitor can either sign up (1) or not (0).

  • The quantity we care about is θ = πA − πB, the difference in sign-up rates between the two CTAs.

  • Our estimator of θ is the difference in sample means, θ̂ = X̄A − X̄B.

Simulating Data

Note

  • Write a brief paragraph explaining that in the real world we would not know the true values of πA and πB, but for this exercise we set them ourselves so we can study the behavior of our estimator.

  • Suppose πA = 0.22 and πB = 0.18, so the true difference is θ = 0.04.

  • Using a language of your choice, simulate 1,000 draws from each of the two Bernoulli distributions (2,000 total).
