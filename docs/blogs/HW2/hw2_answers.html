Introduction

In 1992, New Jersey raised its minimum wage while neighboring Pennsylvania did not. David Card and Alan Krueger exploited this policy discontinuity as a natural experiment: fast-food restaurants in eastern Pennsylvania serve as a comparison group for restaurants in New Jersey that faced the same local demand shocks but not the same minimum-wage shock. The authors surveyed stores before and after the policy change and compared employment, wages, and prices across states and across waves.

The paper became famous because it challenged a simple textbook prediction that a higher minimum wage must reduce employment, at least in the short run, in a low-wage, high-turnover industry where firms have some wage-setting power and adjustment margins beyond immediate layoffs. The “NJ vs. PA” comparison is the key identification move: it supplies a plausible counterfactual trend for what would have happened in New Jersey absent the reform.

This post replicates a few of the paper’s headline descriptive facts and the simplest regression specification using the public-use flat file (public.dat) and the published codebook and SAS check program logic.

Data processing

The public-use file is fixed-width ASCII (410 stores). Column locations follow the codebook file in njmin/, and variable definitions follow check.sas (for example, EMPTOT and GAP).

Table 2 (Wave 1 means)

Table 2 reports simple averages in the pre-period. Here are Wave 1 means for FTE employment, starting wage, and the “full meal” price index (medium soda + small fries + entree), split by state.

Table 3 (first three rows, first three columns)

For the “before / after / change” comparison, Card and Krueger summarize state-level averages where the after mean uses all non-missing second-wave employment values (and treats permanently closed stores as having zero employment in wave 2, as in the paper’s discussion of closures).

Row 3’s “change” is computed from the state-level means (equivalently, it matches the difference-in-differences of the two changes).

Table 4, column (i): OLS of ΔFTE on the NJ dummy

This is the simplest “difference-in-differences in regression form” specification:

ΔFTEi = α + β ⋅ NJi + ui

Following the discussion in Card and Krueger (and common replication code), I restrict to restaurants with non-missing employment change and non-missing starting wages in both survey waves.

Something additional: a DiD plot

This figure summarizes the Table 3 comparison visually: Pennsylvania provides the “would have happened anyway” trend between waves, while New Jersey deviates upward in the second wave relative to that benchmark.

def markdown_table(headers: list[str], rows: list[list[str]]) -> str:
    out = []
    out.append("| " + " | ".join(headers) + " |")
    out.append("| " + " | ".join(["---"] * len(headers)) + " |")
    for r in rows:
        out.append("| " + " | ".join(r) + " |")
    return "\n".join(out)

def wave1_table(x: pd.DataFrame) -> str:
    md_rows: list[list[str]] = []
    for state, label in [(0, "Pennsylvania"), (1, "New Jersey")]:
        sub = x.loc[x["STATE"] == state]

        fte = sub.loc[sub["FTE_W1"].notna(), "FTE_W1"]
        wage = sub.loc[sub["WAGE_ST"].notna(), "WAGE_ST"]
        meal = sub.loc[sub["PMEAL_W1"].notna(), "PMEAL_W1"]

        md_rows.append(
            [
                label,
                str(int(fte.shape[0])),
                f"{float(fte.mean()):.2f}",
                str(int(wage.shape[0])),
                f"{float(wage.mean()):.2f}",
                str(int(meal.shape[0])),
                f"{float(meal.mean()):.2f}",
            ]
        )

    md = markdown_table(
        [
            "State",
            "n (FTE)",
            "Wave 1 FTE",
            "n (wage)",
            "Starting wage ($/hr)",
            "n (meal)",
            "Full meal price",
        ],
        md_rows,
    )

    note = (
        "_Note:_ Means use all non-missing values within each state (`na.rm`-style). "
        "Counts differ across columns because some stores are missing individual items."
    )
    return md + "\n\n" + note

def table3_block(x: pd.DataFrame) -> tuple[pd.DataFrame, str]:
    rows = []
    for state, label in [(0, "Pennsylvania"), (1, "New Jersey")]:
        sub = x.loc[x["STATE"] == state]
        before = sub["FTE_W1"].mean()
        after = sub["FTE_W2"].mean()
        change = after - before
        rows.append({"state": label, "wave1_fte": before, "wave2_fte": after, "change_fte": change})

    out = pd.DataFrame(rows)
    pa = out.loc[out["state"] == "Pennsylvania"].iloc[0]
    nj = out.loc[out["state"] == "New Jersey"].iloc[0]

    diff = pd.Series(
        {
            "state": "NJ − PA",
            "wave1_fte": nj["wave1_fte"] - pa["wave1_fte"],
            "wave2_fte": nj["wave2_fte"] - pa["wave2_fte"],
            "change_fte": nj["change_fte"] - pa["change_fte"],
        }
    )

    tbl = pd.concat([out, diff.to_frame().T], ignore_index=True)

    md = markdown_table(
        ["", "Pennsylvania", "New Jersey", "NJ − PA"],
        [
            [
                "Wave 1 FTE",
                f"{float(pa['wave1_fte']):.2f}",
                f"{float(nj['wave1_fte']):.2f}",
                f"{float(diff['wave1_fte']):.2f}",
            ],
            [
                "Wave 2 FTE",
                f"{float(pa['wave2_fte']):.2f}",
                f"{float(nj['wave2_fte']):.2f}",
                f"{float(diff['wave2_fte']):.2f}",
            ],
            [
                "Change in mean FTE",
                f"{float(pa['change_fte']):.2f}",
                f"{float(nj['change_fte']):.2f}",
                f"{float(diff['change_fte']):.2f}",
            ],
        ],
    )

    return tbl, md

Interpretation: the estimated coefficient on the New Jersey dummy is 2.277 FTE employees per store (HC1 robust SE 1.450, n = 351).

This lines up closely with published magnitudes, but it is not always identical to the paper’s printed 2.33 for a 357-store sample when using this ASCII public-use extract: many replications end up with 351 usable rows under a “non-missing wages in both waves” rule.

A before-and-after comparison in New Jersey alone conflates the minimum wage increase with any aggregate trend affecting fast-food employment between early 1992 and late 1992 (seasonality, local demand, macro shocks, supply disruptions, and so on). A Pennsylvania control group is meant to absorb those common shocks under a parallel trends assumption: absent the policy, NJ and PA would have evolved similarly. The difference-in-differences estimand compares changes across states, which removes time-invariant differences between regions and isolates a policy-induced shift relative to a credible counterfactual trend.
